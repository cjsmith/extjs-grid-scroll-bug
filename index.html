Buffer Rendered Grid Bug

Ext JS's grid supports buffered rendering, which enables you to load large numbers of records into a grid without paging.

However, it uses a real div that gets scrolled and that div has a maximum allowable height restriction imposed by the browser, which means that the scroll bar will only scroll to the last pixel of the maximum height, not the last row coming from the store

In the following example, if you drag the scroll bar to the bottom, you should see the 10,000,000th row, but instead you will see only a million or so rows depending on your browser.
